# Employee Management & Workflow System

A comprehensive web application developed with **ASP.NET Core** and **MS SQL Server** to digitize corporate employee management, workflows, and leave processes.

The system is built on two core roles—**Admin** and **Employee**—and aims to manage modern business processes end-to-end.

## 🚀 Key Features

### 🛡️ Admin Dashboard

* **Department Management:** Create, edit, and manage departments according to the company structure.
* **Employee Management:** Add new employees, update existing records, or remove staff from the system.
* **Leave Request Review:** View, approve, or reject leave requests submitted by employees.
* **Task Management:** Assign specific tasks to employees and monitor their progress (Pending, In Progress, Completed) in real-time.

### 👤 Employee Dashboard

* **Task Tracking:** Inspect assigned tasks and update their status as work progresses.
* **Leave Management:** Create new leave requests and track the approval status of past requests.
* **Profile Management:** View and update personal profile information.

## 🛠️ Tech Stack

* **Framework:** ASP.NET Core MVC
* **Database:** Microsoft SQL Server
* **ORM:** Entity Framework Core (Code-First)
* **Authentication:** ASP.NET Core Identity (Role-Based Authorization)
* **Frontend:** Bootstrap, HTML5, CSS3

## ⚙️ Setup & Installation

1. **Clone the Repository:**
   
   ```bash
   git clone https://github.com/UgurSengezer8/HRMS-ASP.Net.git
   ```

2. **Database Configuration:**
   Update the `ConnectionStrings` section in the `appsettings.json` file with your local SQL Server credentials.

3. **Apply Migrations:**
   Create the database and tables using the **Package Manager Console** in Visual Studio:
   
   ```powershell
   Update-Database
   ```

4. **Run the Application:**
   Press `F5` in Visual Studio to launch the project.
